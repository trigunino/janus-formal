# Prochain jalon : couplage induit par la bimétrie

Le modèle actuel montre seulement qu’un terme \(\lambda X_+X_-\) peut produire
de l’intrication. Pour tester une idée Janus, \(\lambda\) doit provenir d’une
structure bimétrique.

## Construction minimale

Introduire deux métriques de fond diagonales :

\[
g_+=\mathrm{diag}(1,-1),\qquad
g_-=\mathrm{diag}(1,-c^2).
\]

Utiliser leur invariant de désaccord comme paramètre de couplage :

\[
\lambda = \alpha\,\bigl|\mathrm{Tr}(g_+^{-1}g_-)-2\bigr|.
\]

Ce choix est provisoire : il sert uniquement à vérifier si une différence de
métriques peut remplacer le couplage arbitraire.

## Tests obligatoires

1. \(g_+=g_-\) doit donner \(\lambda=0\).
2. Une variation de \(c\) doit modifier les corrélations continûment.
3. L’évolution doit rester unitaire.
4. Le modèle doit respecter la non-signalisation.
5. Il faut comparer avec un modèle à un seul secteur ayant le même nombre de
   paramètres libres.

## Interprétation

Même un résultat positif ne montrerait pas que la gravité crée l’intrication.
Il montrerait seulement qu’un couplage dérivé d’un invariant bimétrique peut
reproduire une famille d’états intriqués. La prochaine exigence serait alors
une action covariante et une limite faible permettant de relier \(\alpha\) à
une constante physique mesurable.
