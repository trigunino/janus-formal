# Prédictions potentiellement testables

Le modèle ne devient physique que s’il relie le rapport cosmologique
\(r=a_-/a_+\) à une quantité mesurable dans notre secteur.

## Observable jouet

Pour deux systèmes quantiques préparés dans un état produit, on calcule

\[
C(r,t),\qquad S_{CHSH}(r,t),
\]

avec le couplage issu du potentiel bimétrique. Une signature candidate serait
une correction de corrélation dépendant de \(r\), impossible à absorber dans
un simple changement de fréquence locale.

## Comparaison obligatoire

À chaque valeur de \(r\), comparer :

1. le modèle bimétrique ;
2. un Hamiltonien standard ajusté pour avoir la même énergie locale ;
3. les données expérimentales disponibles.

Si les modèles 1 et 2 sont indistinguables, la bimétrie n’a aucune signature
quantique propre dans ce secteur.

## Limite cosmologique

Il faudrait ensuite obtenir \(r(t)\) depuis les équations cosmologiques, puis
propager cette valeur jusqu’à une expérience locale. Sans cette chaîne,
\(C(r,t)\) reste une prédiction interne au modèle jouet.

## Critère de réussite

Une correction non dégénérée avec les paramètres standards, compatible avec
la causalité et les bornes expérimentales, serait nécessaire avant de parler
d’une implication pour notre Univers.
