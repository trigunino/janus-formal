# Donnees

Ce dossier est reserve aux jeux de donnees publics.

Ne pas committer de gros fichiers bruts si le projet devient partage. Preferer :

- un script de telechargement ;
- une URL source ;
- une somme de controle ;
- une petite version derivee si necessaire.

Premieres cibles :

- Pantheon+ : https://zenodo.org/records/16365279
- DESI DR2 : https://www.desi.lbl.gov/2025/03/19/desi-dr2-results-march-19-guide/
- KiDS-1000 cosmic shear : https://kids.strw.leidenuniv.nl/DR4/KiDS-1000_cosmicshear.php

Les fichiers bruts telecharges par les scripts sont places dans `data/raw/`, ignore par Git.
