# Lean Workspace

Lean is installed in the current environment through `elan`.

Run:

```powershell
lean formal\lean\JanusBasic.lean
```

This folder is intentionally small. Do not start by formalizing tensor calculus.
Use Lean first for stable logical kernels:

- two sectors;
- same-sector vs opposite-sector cases;
- simple consequences of declared axioms.

Tensor/geodesic formalization should come after the LaTeX and SymPy layers are stable.
