# Janus Complex Reality Candidate Phase-Space Matrix

## Rule

A candidate is useful only if it supplies a physical boundary phase-space
two-cycle with nonzero KKS period.

Topology alone is not enough.

## Core Candidates

| Candidate | Object | Strength | Missing | Status |
| --- | --- | --- | --- | --- |
| Active throat `S2` | geometric `S2` on `Sigma` | already present in the throat geometry | active `Omega_Sigma` pullback and nonzero area component | best direct route |
| `CP1` spinor/frame orbit | `SU(2)/U(1) ~= S2` | minimal compact KKS orbit; period `4*pi*j` | prove Janus/PT requires this orbit and derive `j -> alpha` | best quantum candidate |
| `aroundSigma x phase` | `S1 x S1` action-angle torus | uses existing `aroundSigma` one-cycle | derive compact physical phase and nonzero cross KKS term | plausible but higher risk |

## Extensions

| Extension | Base Candidate | Could Help If | Not Useful If |
| --- | --- | --- | --- |
| Moebius/twisted throat | `aroundSigma x phase` | twist derives a compact phase or Pin/spin holonomy with nonzero KKS period | it only changes visual/topological interpretation |
| Klein/twofold resolved shadow | `aroundSigma x phase` | quotient/lift creates a physical compact phase direction | it only restates the `Z2` cover |
| Higher `CPn` / flag orbit | `CP1` | Janus/PT naturally has a larger internal compact orbit | primitive periods reduce to `CP1` roots anyway |
| Spin/Pin lift around throat | `CP1` or `aroundSigma x phase` | lift fixes a spinor orbit or quantized holonomy | no nonzero boundary symplectic form is produced |
| TQFT/level on `Sigma` | `CP1` or torus | level is derived from Janus action/boundary data | level is inserted by hand |
| Casimir/topological vacuum | active throat `S2` | produces a source and symplectic state law tied to `Sigma` | only gives energy scale, not KKS two-cycle |

## Decision

Extensions are worth opening only if they produce one of:

1. a nonzero active `Omega_Sigma` on the throat `S2`;
2. a Janus-derived `CP1` spinor/frame orbit;
3. a compact phase paired with `aroundSigma`.

If an extension does not do one of these, it cannot fix `alpha` in this branch.
