# Radical Quantum Geometry Reconstruction Parking

## Idea

Replace the classical `Sigma` throat support by a more fundamental quantum
projective/twistor-like geometry.

In that branch, `Sigma` would be emergent rather than primary:

```text
quantum/projective phase space -> emergent throat geometry
```

## Status

Opened as a distinct audit branch.

This is not the same as the previous quantum-first branch:

- previous branch: quantize or fiber an already classical `Sigma/PT` throat;
- radical branch: derive `Sigma/PT`, topology, metric scale and `alpha` from a
  primary quantum state space.

Current verdict:

`not_previously_fully_tried_but_not_closeable_from_current_assets`.

Required maps:

1. primary quantum state space;
2. emergent topology map to `S4/RP4/PT bridge`;
3. emergent metric map to `a(u)`, `R_s`, `g_plus/g_minus`;
4. mass/charge operator whose spectrum gives `M_bridge` and `alpha`.

Without those maps, this is a possible new theory direction, not a closure of
the current Janus model.

## Bottom Verdict

Audited candidate families:

- spin-network / area-spectrum route;
- Chern-Simons or TQFT boundary route;
- twistor / complex-projective route;
- noncommutative spectral-triple route;
- group-field / tensor-model route;
- Souriau coadjoint-orbit route.

All can motivate parts of the problem, but none currently derives the full chain:

```text
primary quantum state/action
-> S4/RP4/PT bridge topology
-> semiclassical bimetric metric map
-> Hamiltonian or charge operator
-> alpha eigenvalue
-> published Janus classical limit
```

Status: future new-theory program, not a closure of the present Janus model.
