# Programme B — intégration des nouveaux résultats de P

Deux dépendances de B sont maintenant réduites.

1. **Transport matière.** P prouve localement en coordonnées arbitraires
   l'identité scalaire `div T = E_phi sharp(d phi)` à partir du jet de
   Levi-Civita. Le point 4 de B possède donc une réalisation matière locale,
   et plus seulement une conséquence abstraite de l'invariance de l'action.
2. **Type causal du throat.** Le normal canonique de P a un carré `+1`.
   La jonction canonique est non nulle/spacelike; l'ambiguïté de normalisation
   des joints nuls est retirée de ce cas précis.
3. **Recollement GHY.** P fournit désormais un atlas local lisse du normal
   canonique avec cocycle de signe. La combinaison orientée `epsilon*K` est
   invariante sur les overlaps; aucun normal global interdit n'est requis.
4. **Racine globale diagonale.** Les métriques diagonales positives lisses de
   P possèdent une racine principale lisse globale. Le point 1 de B est fermé
   sur ce sous-secteur, sans budget d'évolution supplémentaire.

P ne fournit pas encore : une histoire FLRW `H_±(t)`, une charge scalaire
canonique complète pour `r_V`, une action de matière croisée signée, ni une
borne globale sur l'évolution de `g_+^{-1}g_-`. Les verdicts globaux des points
1, 6 et 7–8 ne changent donc pas encore.
