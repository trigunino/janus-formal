# Programme B — coefficient cubique Lambda_3

Dans les conventions de la limite de découplage de Noller–Scargill
(arXiv:1503.02700, équations 610–611), la contribution cubique duale est

`E3 = betaHat1 ([Pi]²-[Pi²])`
`   +(3/2) betaHat2 ([Pi²]-[Pi]²)`

donc

`E3=(betaHat1-3 betaHat2/2) U2(Pi)`.

Pour une configuration statique sphérique, les valeurs propres de `Pi` sont
`lambda_r=pi''` et deux fois `lambda_t=pi'/r`, d'où

`U2=2 lambda_t(2 lambda_r+lambda_t)`.

Après une intégration radiale, ce terme produit bien l'équation quadratique
utilisée dans le profil réduit de Vainshtein. La surface
`betaHat1-3 betaHat2/2=0` supprime le mécanisme cubique.

Il reste à établir la carte de normalisation entre les `betaHat` de cette
convention, les deux contributions bigravity et les `beta_n,K_plus,K_minus`
de Candidate A. Cette séparation évite d'attribuer un signe physique à une
combinaison dépendante des conventions.
