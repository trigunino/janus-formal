# Programme B — carte beta vers les mélanges Lambda_3

L'expansion directe de

`sum_n (-1)^n beta_n Y_n(I+Pi)`

dans la base conservée `X^(n)(Pi)` donne exactement

- `alpha1=beta1+2 beta2+beta3`;
- `alpha2=beta2+beta3`;
- `alpha3=beta3`.

Les quatre composantes diagonales symboliques vérifient l'identité.

Sur la branche PT `beta3=beta1`, `beta1>0`, `beta2>=0` :

`alpha1=2(beta1+beta2)>0`,

`alpha2=beta1+beta2>0`,

`alpha2/alpha1=1/2`.

Le mélange qui génère le Galileon cubique ne s'annule donc pas sur le cône PT;
un mécanisme de Vainshtein cubique est bien présent dans cette convention.
Il reste à combiner les deux cadres du bigravity et à canoniser le scalaire
pour obtenir le coefficient radial absolu, pas seulement son existence et son
rapport de forme.
